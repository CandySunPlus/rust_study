# simple rust study project

## Build

```zsh
$ cargo build --release
```

## Demo List

### guess-number

```zsh
$ cargo run guess-number
```
### dining-philosophers

```zsh
$ cargo run dining-philosophers
```
### embed

```zsh
$ cd ./src/embed/
$ npm i
$ cd -
$ node ./src/main.js # use rust embed library
$ node ./src/pure.js # use nodejs cluster
```
